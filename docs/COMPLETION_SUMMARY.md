## ⭐ 亮点功能

### 1. 两种数据提交方式
**创新点**：用户可以通过 Git push 或 GitHub Issue 提交数据
```
Git Push   → 开发者友好
Issue      → 非技术用户友好
两者并行   → 最大化贡献
```

### 2. 完全自动化的处理流程
**创新点**：Issue 提交后完全自动处理，无需人工审核
```
Issue 创建 → 自动解析 → 自动验证 → 自动更新 JSON
→ 自动提交 → 自动关闭 → 全过程自动化
```

### 3. 智能变化检测
**创新点**：基于哈希值的变化检测，避免不必要的重绘
```
每 30 秒检查 → 计算哈希值 → 与上次比对
→ 变化才更新 → 高效的资源使用
```

### 4. 双光污染等级系统
**创新点**：同时支持国际和中国标准
```
Bortle (国际) + Standard Light (中国) = 完整信息
```

### 5. 中心浮动式信息面板
**创新点**：相比传统侧边栏更适合移动设备
```
中心浮动 → 不遮挡内容 → 移动友好
动画效果 → 视觉体验好 → 专业感
```
