## 📋 执行摘要

### 项目概述
设计并实现了一个**静态网页地图应用**，展示观星地点，支持：
- 高德地图集成与自定义标记
- 自动检测 JSON 数据更新
- 完全自动化的 GitHub Issue 数据提交系统
- 光污染等级映射与颜色编码
- 响应式设计（桌面/平板/手机）
